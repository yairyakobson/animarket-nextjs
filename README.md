This is Animarket built in NextJS App Router.

Legacy build (archived) https://github.com/yairyakobson/animarket

To run the application run the **npm run build** command and in a separate terminal run:
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Or:
```bash
npm run start
# or
yarn start
# or
pnpm start
# or
bun start
```
